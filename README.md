# Stock Buster

## Description
Outil de gestion de stock pour les entreprises.

## Installation

### Prérequis

- NODEJS 20.18.0
- NPM 10.8.2
- MONGO
- DOCKER
- DOCKER-COMPOSE
- GIT

### Installation

1. Cloner les projets
```bash
make init
```
### Lancer le projet
```bash
make start
```

### Arrêter le projet
```bash
make stop
```

### Update le projet
```bash

make update
```

### Voir les logs
```bash
make logs
```
